# health

## サイト概要
健康・ダイエット情報共有サイト

### サイトテーマ
人生100年時代、健康で美しく、心身ともに健やかに！

### テーマを選んだ理由
現代は人生100年時代と呼ばれ、健康で生きることの価値は偉大である<br>
健康は日々の習慣から生まれる<br>
気軽に共有できる場ががあると良いと思い、選定<br>
健康への意識が高まり、心身ともに健やかな方々が増えると嬉しい<br>

### ターゲットユーザ
健康志向、ダイエットに関心のある方

### 主な利用シーン
健康、ダイエット情報の取得、発信
